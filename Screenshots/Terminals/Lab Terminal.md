#terminal 
When in [[The Lab]] you can access this [[terminal]]
- It has 5 options
	1. Experiment Setup
		-  Allows you to start a new [[Lab Experiments]]
	2. Research Logs
		-  Shows me every [[Lab Experiments]] I have ever done!
	3. Pause Experiment
		-  Lets me temporarily stop the current day's [[Lab Experiments]]
	4. Packet Management
		-  Lets me pick what effects/triggers can spawn.  Must have a minimum of 8 for each.
	5. Login To Network
		- Allows you to see the standard network options with the password "Swansong"
		- See [[Shared Terminal]]
