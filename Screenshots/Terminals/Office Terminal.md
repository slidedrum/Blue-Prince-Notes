#terminal 
When in [[The Office]]you can access this [[terminal]]
- It has 3 options
	1. Run Payroll
		- Add money to all servants room's
			- [[The Servant's Quarters]]
			- [[The Maid's Chamber]]
			- and probably others.
	2. Spread Gold in Estate
		- [[Spread]] [[Gold (Coins)|gold]] into most rooms of the estate.
	3. Login to Network
		- Allows you to see the standard network options with the password "Swansong"
		- See [[Shared Terminal]]
